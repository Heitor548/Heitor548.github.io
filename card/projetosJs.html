App Orçamento Pessoal javascript

Bem, a aplicação desenvolvida "App Orçamento Pessoal" é um ambiente onde o usuario pode cadastrar suas despesas, informando data completa, descrição e o valor daquela despesa. Posteriormente fazer consultas e remover determinada despesa. Faz uso do recurso de local storage do navegador que atuará como um "banco de dados".

Calculadora javascript

Calculadora básica usando funções para realizar as operações. Ela funciona por meio do metodo "onclick" e envia parametros (valor, tipo da operação). Trabalha um pouco do manejo do DOM.

Simulador de Pandemia javascript

O simulador de pandemia foi um projeto desenvolvido para disputa de feira tecnica. Com o intuito de mostrar as pessoas os efeitos das medidas preventivas no mundo durante uma pandemia. Foi desenvolvido com base em uma API do google a chamada "google charts".
